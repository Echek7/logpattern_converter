# LogPattern Converter CLI

## 🛠️ Herramienta Profesional de Refactorización de Logs

LogPattern Converter es una herramienta de línea de comandos (CLI) esencial para ingenieros de DevOps y administradores de sistemas que necesitan **refactorizar, convertir y estandarizar patrones de logs complejos** de entornos heredados (legacy) a formatos modernos como JSON o estándares como OpenTelemetry.

Elimine las horas de trabajo manual en la migración de logs.

### 🚀 Funcionalidades Clave
* **Conversión Automática:** Identifica patrones de logs y los traduce a esquemas estandarizados.
* **Validación de Formato:** Asegura la compatibilidad con plataformas modernas de observabilidad.
* **Fácil de Usar:** Interfaz de línea de comandos simple y rápida.
